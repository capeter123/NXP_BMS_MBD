Block Name

Memory Read Block

Block Description

The main functionality of the block is to read data at memory location specified by the base address and the offset value. That is reading data at memory location (Base Address + Offest.)

Block Image

Inputs:

  • None

Outputs:

  • 32bit Data at the specified 32bit memory location.

Block Dialog and Parameters:

Names Selection Types Range Description
Enter base Address Text-box 32bit Hex value
Enter Offset from the base address Text-box 32bit Hex value

Block Dependency

None

Block Miscellaneous Details:

The block requires that the memory address (base address + offset) must be within 32bit boundary.
