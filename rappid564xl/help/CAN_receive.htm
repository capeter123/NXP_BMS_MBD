Block Name

FlexCAN Receive Data Block

Block Description

The main functionality of the block is receiving data through the FlexCAN interface.
The block waiting till message will come.

Block Image

Inputs:

  • None

Outputs:

  • ID (uint8)
  • ID Type (uint8)
  • Message (uint8(8))
  • Time stamp (unit16)

Block Dialog and Parameters:

Names Selection Types Range Description
CAN Bus Pop-up 0 or 1 CAN Module
Buffer Pop-up 0 – 31 Buffer Number
ID Text-box ID of the message to receive

Block Dependency

Please do the following:
  1. Configure FlexCAN

Block Miscellaneous Details:

Please refer "CAN Configiration block" to get information about pin assignment.
