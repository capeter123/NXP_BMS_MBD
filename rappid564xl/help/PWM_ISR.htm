Block Name

FlexPWM ISR Block

Block Description

The main functionality of the block is to process FlexPWM ISRs

Block Image

Inputs:

  • None

Outputs:

  • Function-call
  • Status register (STS) value (uint16) (if "Interrupt source" is set to "Compare interrupt")
