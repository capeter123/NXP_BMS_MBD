Block Name

DSPI Interrupt Enable/Disable Block

Block Description

The main functionality of the block is to enable/disable DSPI interrupts

Block Image

Inputs:

  • None

Outputs:

  • None

Names Selection Types Range Description
DSPI Pop-up 0,1,2 DSPI Module

Enable RFOF Check-box On/Off Enable/Disable RFOF interrupt
Enable EOQF - End of Queue Check-box On/Off Enable/Disable EOQF interrupt
Enable TFFF - Transmit FIFO Full Check-box On/Off Enable/Disable TFFF interrupt
Enable TCF - Transfer Complete Check-box On/Off Enable/Disable TCF interrupt
Enable RFDF - Rx FIFO Drain Check-box On/Off Enable/Disable RFDF interrupt

Block Dependency
