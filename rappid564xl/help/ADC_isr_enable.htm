Block Name

ADC Interrupt Enable/Disable Block

Block Description

The main functionality of the block is to enable/disable ADC interrupts

Block Image

Inputs:

  • None

Outputs:

  • None

Block Dialog and Parameters:

The block dialog consists of the following tabs:
  • ADC ISR
  • WDG CH(0-7)
  • WDG CH(8-15)
  • ADC Channels
  • The ADC ISR tab contains the following parameters:
    •
Names Selection Types Range Description
ADC converter (0 - 1) Pop-up 0 – 1 Select the ADC converter number
Enable EOCTU interrupt Check-box On/Off Enable/Disable EOCTU interrupt
Enable JEOC interrupt Check-box On/Off Enable/Disable JEOC interrupt
Enable JEGH interrupt Check-box On/Off Enable/Disable JEGH interrupt
Enable EOC interrupt Check-box On/Off Enable/Disable EOC interrupt
Enable EGH interrupt Check-box On/Off Enable/Disable EGH interrupt
  • The WDG CH(0-7) tab contains the following parameters:
    •
Names Selection Types Range Description
Enable WDG CHn Low Check-box On/Off Enable/Disable WDG CHn Low interrupt (n is 0 – 7)
Enable WDG CHn High Check-box On/Off Enable/Disable WDG CHn High interrupt (n is 0 – 7)
  • The WDG CH(8-15) tab contains the following parameters:
    •
Names Selection Types Range Description
Enable WDG CHn Low Check-box On/Off Enable/Disable WDG CHn Low interrupt (n is 8 – 15)
Enable WDG CHn High Check-box On/Off Enable/Disable WDG CHn High interrupt (n is 8 – 15)
  • The ADC Channels tab contains the following parameters:
    •
Names Selection Types Range Description
Enable CHn Check-box On/Off Enable/Disable CHn interrupt (n is 0 – 15)

Block Dependency

Please do the following:
  1. Configure related ADC chanel and its interrupts
